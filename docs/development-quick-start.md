# Development Quick Start

## Requirements

- Node.js 18+ (recommended: Node.js 20)
- npm (bundled with Node.js)

## Install

```bash
npm install
```

## Daily commands

```bash
npm run dev
npm run build
npm run lint
npm run scan
npm run test
```

## Suggested validation before PR

```bash
npm run lint
npm run scan
npm run build
```

## Notes

- This repository is standardized on **npm scripts**.
- If you previously used Yarn commands, use their npm equivalent commands above.
